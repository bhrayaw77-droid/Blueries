Tasks are saved in localStorage so they remain even after refreshing the page. Your work is always preserved, automatically.

Learning Purpose

Built to Demonstrate
Core Web Concepts

This project is a hands-on demonstration of fundamental JavaScript and browser API concepts, designed to show how modern interactive web applications are built from scratch.

  • ⚙️
    DOM Manipulation Dynamic rendering and real-time UI updates with JavaScript
  • 🖱️
    Drag and Drop Events HTML5 native drag-and-drop API for seamless interactivity
  • 🗂️
    Application State Managing and synchronizing UI state across components
  • 🔒
    localStorage Persistence Storing and retrieving data across browser sessions
kanban.js 
// Save tasks to localStorage
function saveTasks(tasks) {
  localStorage.setItem(
    'kanban-tasks',
    JSON.stringify(tasks)
  );
}
